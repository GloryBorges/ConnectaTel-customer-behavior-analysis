# ConnectaTel-customer-behavior-analysis
Análisis avanzado de datos de clientes de ConnectaTel en Latinoamérica. Este proyecto explora patrones de uso, segmenta clientes por edad y nivel de consumo, detecta outliers y genera insights accionables para mejorar la retención y optimizar los planes de telecomunicaciones.
**tecnologias y librerias**

Python 3.x

Pandas, NumPy

Matplotlib, Seaborn

Jupyter Notebook
